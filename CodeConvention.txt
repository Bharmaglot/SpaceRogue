GIT
1. Основная ветка - DI
2. Название создаваемой ветки - "Раздел"/"Автор"/"Задача".
Например, для задачи "Босс Болванка" создаётся ветка Enemy/VascoDaKrava/Boss
Варианты разделов: Enemy, Level, Menu, Ship

3. Все коммиты должны быть с комментариями, что сделано
4. Пул-реквест также должен содержать описание



CODE
1. Кириллица в коде запрещена. Язык всего - американская версия английского.

2. Для скриптов использовать "namespase SpaceRogue.*".

3. Linq использовать минимально.

4. Код оборачивать в регионы:
Events - для событий
Fields - для полей (стараться делать приватными)
Properties - для свойств (стараться делать публичными)
Mono - для функций Unity: Awake, Update, OnDisable и т.д.
CodeLife - для конструкторов, деструкторов и реализации IDisposable
Methods - для методов
Enums/Structs - для локальных ресурсов

5. Всё публичное располагается выше приватного по коду.

6. Использовать наименьшую видимость модификаторов доступа public/private.

7. Имена использовать передающие суть, состоящие только из букв и подчёркивания.
Регистр:
_privateVariable - приватные
localVariable - локальные для метода
PublicName - публичные и имена классов, методов и т.д.
SOME_CONSTANT - константы

8. "Магические" числа, строки запрещены.

9. Сокращения запрещены.

10. Строки выводить через интерполяцию - $"Variable {_myVariable}"

11. Float (и другие дробные) всегда использовать с точкой и указанием соответствующей буквы ("f" для флоата).
Например:
0.5 ==> .5f или 0.5f
1 ==> 1.0f

12. Пробелы и переносы строк использовать как в правилах русского/английского языка.
Орфографию и пунктуацию также необходимо соблюдать.

13. Все исползуемые пространства имён должны быть импортированы и отсортированы по алфавиту.

14. Использовать var.

15. Не использовать папку Resources.
https://learn.unity.com/tutorial/assets-resources-and-assetbundles#5c7f8528edbc2a002053b5a6

16. Разделять участки кода пустыми строками.

17. Использовать форматирование документа (для VisualStudio - ctrl+k+d).

Х. Ознакомиться с https://docs.google.com/document/d/1FNUaiOCvPLDRXX1nKfba42wUvN2EpunkX3JHoZ4_szM
